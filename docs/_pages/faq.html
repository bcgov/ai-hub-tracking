Why does the AI Hub need larger VNets than standard /24?

Answered

The AI Hub will run multiple Azure services that each require significant IP allocation:

Service Min Size Recommended
API Management (APIM) /26 /24
App Gateway /26 /24
AI Foundry /25 /24
PrivateLink Subnet /27 /27
AKS Variable Variable

Total: This exceeds the standard /24 (256 IPs), requiring at least a /22 for production.

Technical Architecture

Why can't GitHub Actions run Terraform directly?

Answered

See ADR-001: Shared Landing Zone for the full explanation.

Short answer: BC Gov policy requires all Azure services to use private endpoints. GitHub-hosted runners are on the public internet and cannot reach private endpoints. Self-hosted runners inside the VNet are required.
